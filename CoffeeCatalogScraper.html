Web Scraping: World Coffee Research

Part of a larger project to build a holistic coffee database containing everything from origin information about different varietals to individual cupping scores, this web scraper was used to parse the World Coffee Research public catalog of coffee varietals. The script first parses coffee names from the original list, then serializes each coffee's data into a structure to write into an SQL table.
